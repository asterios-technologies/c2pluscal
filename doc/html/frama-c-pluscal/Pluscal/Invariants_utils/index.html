type entry =
  1. | Local of local_entry * string * int
  2. | Global of global_entry * string * int

Type representing a configuration entry of a .expect file Can be either local or global, in a tuple with the process name and id, in which the check is performed *

val entry_of_json : Yojson__Safe.t -> (entry, string) Stdlib.result

Parses a JSON object to extract configuration details and returns a result containing either a Global or Local entry, or an error message.

  • parameter json

    JSON object to parse.

  • returns

    Result containing entry parsed of JSON file or an error message.

The JSON object is expected to have the following fields:

  • "var_name": String representing the variable name.
  • "expected_val": String representing the expected value.
  • "proc_to_check": Tuple containing a string (process name) and an integer (process ID).
  • "proc_of_var": (Optional) Tuple containing a string (process name) and an integer (process ID) for local entries. *
val parse_expect_file : string -> (entry, string) Stdlib.result list

Parses a JSON file containing configuration entries. The JSON file is expected to contain a list of entries.

  • parameter filename

    path to the JSON file to be parsed.

  • returns

    entry List parsed from the JSON file.

  • raises [Json_error]

    if the JSON file does not contain a list of entries or if there is an error parsing the file. *
