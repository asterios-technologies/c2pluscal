Module Pluscal.Config_utils

Utilities for generating configuration invariants. *

type entry = {
  1. var_name : string;
  2. proc_name : string;
  3. proc_id : int;
  4. expect_val : string;
}

Type representing a configuration entry of a .expect file

  • parameter var_name

    name of the variable.

  • parameter proc_name

    name of the process.

  • parameter proc_id

    id of the process.

  • parameter expect_val

    expected value for the variable. *

val entry_of_json : Yojson__Safe.t -> entry

Converts a JSON object into a configuration entry

  • parameter json

    JSON object to be converted.

  • returns

    entry extracted from the JSON object. *

val parse_expect_file : string -> entry list

Parses a JSON file containing configuration entries. The JSON file is expected to contain a list of entries.

  • parameter filename

    path to the JSON file to be parsed.

  • returns

    entry List parsed from the JSON file.

  • raises [Json_error]

    if the JSON file does not contain a list of entries or if there is an error parsing the file. *
